# Output Formats

Choose the appropriate format based on the audience and content type.

## Format Options

### Executive Summary
Use when the audience needs a quick overview. Structure:
- 3-5 sentences covering the most important findings
- Lead with the single most significant conclusion
- End with any required actions or decisions

### Key Findings
Use for stakeholders who need actionable details. Structure:
- Bulleted list of findings, ordered by importance
- Each bullet is one complete thought (1-2 sentences)
- Group related findings under subheadings if more than 7 items

### Detailed Breakdown
Use when thoroughness matters more than brevity. Structure:
- Section-by-section summary following the original structure
- Each section gets a heading, 2-4 sentence summary, and notable data points
- Include cross-references between related sections

### Data Tables
Use when the input is primarily quantitative. Structure:
- Markdown tables preserving key metrics
- Column headers match the original data labels
- Include totals, averages, or ranges where meaningful
- Add a 1-2 sentence interpretation below each table

## Choosing a Format

- **Default to Key Findings** when no format is specified.
- Use Executive Summary for reports, meeting notes, and long-form documents.
- Use Data Tables for spreadsheets, logs, and metrics.
- Combine formats when the input has both narrative and quantitative content.
