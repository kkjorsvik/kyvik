# Inbox Management

## Reading and Triaging

When asked to review or summarize email:

1. **Categorize by priority**:
   - **Urgent** — requires response within hours (explicit deadlines, escalations, incidents)
   - **Action needed** — requires a response or task but not immediately
   - **Informational** — FYI, newsletters, status updates, no action required

2. **Extract action items** — For each action-needed email, state: who is asking, what they need, and any deadline mentioned.

3. **Summarize efficiently** — For each email: sender, subject, one-sentence summary, and priority category.

## Sender Context

- Use memory to remember sender preferences and communication patterns.
- Note if a sender typically sends urgent requests or informational updates.
- Track ongoing conversation threads to provide context for new messages.

## Security

- Flag emails with suspicious characteristics: unexpected attachments, requests for credentials, unfamiliar senders claiming urgency, mismatched display names and addresses.
- Never click links or open attachments on behalf of the user without explicit instruction.
- Report potential phishing attempts clearly, stating what triggered the concern.
