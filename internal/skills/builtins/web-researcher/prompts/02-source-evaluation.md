# Source Evaluation

## Credibility Assessment

For each source, evaluate:

1. **Source type** — Primary sources (original data, official documents, direct statements) are stronger than secondary sources (news articles, summaries, commentary). Tertiary sources (encyclopedias, aggregators) are useful for overview but should not be cited as authoritative.
2. **Publication date** — Prefer recent sources for time-sensitive topics. Note the publication or last-updated date. Flag sources older than 1 year for fast-moving topics (technology, policy).
3. **Author credentials** — Check if the author has relevant expertise or authority on the topic. Institutional affiliations, professional background, and publication history add credibility.
4. **Publisher reputation** — Established institutions, peer-reviewed journals, and recognized news organizations carry more weight than personal blogs or anonymous sites.
5. **Bias indicators** — Check for obvious bias: funding sources, political alignment, commercial interests, or advocacy positions. Biased sources can still contain useful information but should be noted as such.

## When Sources Disagree

- Present both positions clearly.
- Note the credentials and potential biases of each side.
- Identify the strongest evidence for each position.
- Do not resolve the disagreement by choosing a side unless the evidence is overwhelming.
- State what additional information would help resolve the disagreement.

## Red Flags

Flag sources that exhibit:
- No author attribution or anonymous publication
- No publication date
- Extraordinary claims without evidence
- Circular sourcing (sources citing each other with no original data)
- Monetization that creates incentive to mislead (affiliate links, sponsored content)
