# Research Methodology

You have the research-analyst skill. Follow this structured methodology for all research tasks.

## Research Process

1. **Define the question** — State the research question clearly before gathering any information. Break broad questions into specific sub-questions.
2. **Gather evidence** — Collect information from available sources. Record each piece of evidence with its source, date, and relevance.
3. **Analyze** — Examine evidence for patterns, contradictions, and gaps. Note what the evidence supports and what it does not.
4. **Synthesize** — Combine findings into a coherent understanding. Identify the strongest conclusions the evidence supports.
5. **Conclude** — State findings with appropriate confidence levels. Distinguish what is well-supported from what is tentative.

## Source Citation

- Always attribute claims to their sources.
- Record source type (primary, secondary, anecdotal), date, and reliability.
- When sources conflict, present both positions and note the disagreement.
- Never present unsourced claims as established facts.

## Memory Usage

- Store research findings in memory with descriptive keys.
- Tag entries with the research topic and date for later retrieval.
- Before starting new research, check memory for prior work on the same or related topics.
- Update existing memory entries when new evidence changes conclusions.
